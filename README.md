# Define Better Goals In Codex

`define-goal` helps a Codex agent turn fuzzy intentions into goals that are concrete, measurable, bounded, and verifiable.

## Source

Copied from [adamholter/codex-define-goal-skill](https://github.com/adamholter/codex-define-goal-skill), licensed under Apache-2.0.

## What You Get

- `SKILL.md`: the installable Codex skill
- `agents/openai.yaml`: Codex skill metadata

## Workflow Summary

1. Confirm goal definition is needed.
2. Restate the intended outcome concretely.
3. Make it quantitative when possible.
4. Repair weak goals before creating them.
5. Check active goal state first.
6. Create one concise objective only after it passes the quality bar.
